# ML_rec_sys-fast_api
Модели машинного обучения и сервис на базе Fast API для рекомендательной системы.

# Проект рекомендательной системы

В этом проекте реализована рекомендательная система, которая прогнозирует, какие публикации понравятся пользователям, на основе характеристик их профиля, прошлых взаимодействий и содержания публикаций.

## Особенности проекта

- **Первая модель:** построена на основе эмбеддингов публикации, созданных с помощью модели DistilBERT. Модель обучается на целевых данных и учитывает временные характеристики, текстовые данные и статистику взаимодействий.
- **Вторая модель:** в проекте также используется альтернативная модель кластеризации текста на основе РСА и метода k-means, которая добавляет дополнительные характеристики, такие как расстояние до центров кластеров и принадлежность к кластеру.
- **A/B-тестирование:** реализован API-сервис для проведения A/B-тестирования, который позволяет сравнивать эффективность двух групп пользователей.

## Используемые технологии

- **Машинное обучение:** Scikit-learn, PyTorch, Transformers (Hugging Face)
- **Базы данных:** PostgreSQL
- **API:** FastAPI
- **Обработка данных:** Pandas, NumPy
- **Визуализация:** Matplotlib, Seaborn
